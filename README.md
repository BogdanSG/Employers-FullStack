# Employers-FullStack  
  
# Install  
  
git clone https://github.com/BogdanSG/Employers-FullStack.git  
npm i / composer update  
  
# Connection DB Settings  
  
NodeJS: config.json  
PHP: .env  
  
# DataBase  
  
Import DataBase employers.sql or generate new data NodeJS Only :  
Create DataBase 'employers'  
npm restart  
npm insert  
  
# Run  
  
npm start
